```bash
  --path=/var/www/html/[site-name] --allow-root
```

### プラグインリスト取得
```bash
# 全サイトのプラグインリスト取得（wp-cli必要）
for site in $(find /var/www/html -maxdepth 1 -type d); do
  echo "=== $site ==="
  # wp plugin list --path=$site
done
```

## ⚠️ 想定される問題と対処

### 問題1: ライセンスキー紛失
- **対処**: Elementor アカウントから再発行
- **予防**: ライセンスキーを安全な場所に保管（1Password等）

### 問題2: ドメイン変更によるライセンス無効化
- **対処**: Elementor管理画面でドメイン変更申請
- **予防**: 本番移行前にライセンス変更手順確認

### 問題3: プラグイン互換性問題
- **対処**: プラグインアップデート or 代替プラグイン検討
- **予防**: PHP 8.3互換性事前確認

### 問題4: API連携エラー
- **対処**: APIキー再発行、ドメインホワイトリスト更新
- **予防**: 外部API連携プラグインリスト作成
